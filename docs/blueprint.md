# **App Name**: GeoConnect

## Core Features:

- User Authentication and Profiles: Secure user signup, login, profile creation with picture uploads, and editing using Firebase Authentication and Firestore.
- Real-time Interactive Map: Display user's current location, nearby posts, and events on an interactive map with custom markers and clustering using Google Maps API and Firestore.
- Location-Based Social Posts: Allow users to create and share text/image posts with location tags, discover nearby posts, and interact via likes and comments. Data stored in Firestore.
- Event Creation and Discovery: Users can create, join, and discover local events with details, location tagging on the map, and categorized listings.
- Real-time Notifications: Instant notifications for new posts, nearby events, and interactions via Firebase Cloud Messaging.
- Search and Discovery: Find nearby users, posts, and events with filters for distance and categories.
- AI-Powered Content Summarization: Summarize long user bios or event descriptions using a generative AI tool to provide quick insights.

## Style Guidelines:

- Color scheme: A dark mode-focused palette with a primary deep indigo (#4B0082) for depth and trust. Backgrounds are a very dark, desaturated blue (#1A1A2E). Accent color is a vibrant cyan (#00FFFF) for calls to action and highlights, creating a futuristic and tech-forward feel.
- Font pairing: 'Space Grotesk' for headlines to convey a modern, techy feel. 'Inter' for body text for clarity and readability in both long and short contexts.
- Iconography: Clean, minimalist line icons with a subtle glow effect, aligning with the futuristic and glassmorphism design.
- Layout: Employ a clean, card-based design with ample whitespace and glassmorphism effects. Use a bottom tab navigation for core sections of the app.
- Animation: Subtle, fluid animations for screen transitions, list loading, and interactive map elements to enhance user experience without being distracting.
